# BYD Autostart (开机启动流程)

一个专门为 Android 系统（特别是比亚迪车载系统等大屏设备）设计的开机自启动管理工具。它可以帮助您设置在设备启动后，自动按顺序启动多个应用程序，并自定义每个应用之间的等待间隔。

## 功能特性

- **可视化流程**：直观的垂直流程图界面，清晰展示启动顺序。
- **灵活配置**：
  - 支持从已安装的应用列表中选择。
  - 支持 0-30 秒的启动间隔调节（使用 SeekBar 滑动选择）。
  - 支持删除和重新排序。
- **自动退出**：自启动序列执行完毕后，应用会自动从后台任务列表中移除，不占用系统资源。
- **开机自启**：利用 `BOOT_COMPLETED` 广播，确保在系统准备就绪后第一时间执行。

## 使用说明

1. **添加步骤**：点击中间的“添加步骤”按钮，选择您想要在开机后启动的应用。
2. **设置间隔**：通过滑动条设置该应用启动后，需要等待多少秒再启动下一个应用。
3. **保存并测试**：点击下方的“保存配置并测试”按钮。这会保存当前设置并立即模拟一次完整的启动流程。
4. **生效**：配置保存后，下次设备重启时，系统将自动按照此列表执行。

## 开发环境

- **IDE**: Android Studio
- **Language**: Kotlin
- **Min SDK**: 26 (Android 8.0)
- **Target SDK**: 36
- **UI Components**: Material Components (Google)

## 权限说明

- `RECEIVE_BOOT_COMPLETED`: 用于接收系统启动完成的广播。
- `QUERY_ALL_PACKAGES`: 用于列出系统中已安装的应用以便选择。

---

*注意：在某些定制化的 Android 系统（如车载系统）中，可能需要在系统的“自启动管理”或“电池优化”设置中手动允许本应用后台运行或开机自启(对于比亚迪Dilink 3.0/3.5/4.0, 需要在禁止开机自启动和急速模式中添加白名单)。*

*测试车机: 秦PLUS DM-i 2023冠军版, 超越型*
